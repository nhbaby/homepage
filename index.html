AI 쇼츠 팩토리 (VidFactory AI)

프로그램 버전: 1.2.2 (GUI)

🚀 프로그램 소개

'AI 쇼츠 팩토리 (VidFactory AI)'는 인공지능(AI)의 놀라운 힘으로 여러분의 제품이나 아이디어를 소개하는 짧고 멋진 영상(쇼츠)을 쉽고 빠르게 만들어주는 혁신적인 프로그램입니다.

💡 가장 큰 장점: 복잡한 영상 편집 기술이 전혀 없어도 괜찮습니다! 여러분이 제공한 영상과 간단한 설명만 있으면, AI가 알아서 대본을 쓰고, 실감 나는 목소리를 더하고, 필요한 자막까지 척척 만들어 전문가처럼 멋진 쇼츠 영상을 완성해 드립니다.

주요 기능:

  • 📝 AI 자동 대본 & 홍보 문구 생성: 여러분의 아이디어를 AI가 매력적인 영상 대본과 소셜 미디어(유튜브, 틱톡 등) 홍보글로 바꿔줍니다.
  • 🎬 영상 자동 편집: 업로드하신 영상에서 AI가 대본에 딱 맞는 최고의 장면들을 자동으로 골라 자연스럽게 연결해 줍니다.
  • 🔊 고품질 AI 음성 더빙 & 자막: 사람처럼 자연스러운 AI 목소리로 대본을 읽어주고, 내용에 딱 맞는 자막을 자동으로 넣어줍니다.
  • ✨ 사용자 맞춤 설정: 자막의 폰트, 크기, 색상, 위치를 자유롭게 조절할 수 있습니다. 중국어 및 특정 영역을 가리는 '블러' 효과(무료)와 워터마크 추가(유료) 기능도 제공합니다.
  • ⚡ 빠른 영상 제작: 고성능 컴퓨터를 사용하신다면 '하드웨어 가속(GPU)' 기능을 통해 영상 제작 시간을 획기적으로 줄일 수 있습니다.
  • 💻 간편한 사용: 모든 과정은 누구나 쉽게 사용할 수 있는 웹사이트(웹 UI)와 편리한 데스크톱 프로그램(GUI)으로 이루어집니다.

💡 시작하기 (설치 및 최초 실행)

VidFactory AI는 설치 과정이 아주 간단합니다. 대부분의 설정은 프로그램이 자동으로 잡아주거나, 처음 실행 시 나타나는 안내에 따라 몇 가지만 입력하면 바로 사용할 수 있습니다.

  1. GUI 프로그램 실행: 제공받으신 압축 파일을 풀고 ai쇼츠자동생성프로그램.exe 파일을 더블 클릭하여 프로그램을 시작합니다.
  2. (최초 실행) 라이선스 키 입력 🔑: 프로그램을 처음 사용하시거나 라이선스가 만료된 경우, '라이선스 키 입력' 창이 자동으로 뜹니다. 받으신 라이선스 키를 정확히 입력하고 [확인]을 누르면 됩니다. 유효한 키는 만료일과 함께 확인 메시지를 보여주며, 컴퓨터에 안전하게 저장됩니다.
  3. (최초 실행) Google Gemini API 키 입력 🤖: 라이선스 키 입력 후, Google Gemini API 키가 설정되어 있지 않으면 'Google Gemini API 키 입력' 창이 나타납니다. AI 대본 및 홍보 문구 생성에 꼭 필요한 키이니, Google Cloud Console에서 발급받은 API 키를 입력하고 [저장] 버튼을 누릅니다.
  4. 초기 GPU 가속 설정 확인 🚀: 라이선스와 API 키 입력이 끝나면, 프로그램이 자동으로 사용자의 컴퓨터가 '하드웨어 가속(GPU)' 기능을 지원하는지 확인합니다. 'GPU 가속이 활성화되었습니다' 또는 'CPU 모드로 진행합니다' 와 같은 메시지가 나타나며, 별도 조작 없이 [확인] 버튼만 누르면 됩니다.
  5. 메인 제어판 표시: 모든 초기 설정이 완료되면, 'VidFactory AI' 메인 제어판 창이 화면에 나타납니다.

💻 메인 제어판 사용법

메인 제어판은 'VidFactory AI'의 모든 기능을 관리하고, 설정을 바꾸며, 프로그램 상태를 한눈에 파악할 수 있는 중심 허브입니다.

🚀 '프로그램 시작 & 웹 UI 열기' 버튼: 이 버튼을 누르면 'VidFactory AI'의 웹 서버가 켜지고, 인터넷 브라우저가 자동으로 열려 영상 제작 웹사이트로 이동합니다. 서버가 실행 중이면 버튼은 [✅ 프로그램 실행 중]으로 바뀌고 비활성화됩니다. 만약 브라우저가 자동으로 열리지 않으면, 주소창에 http://127.0.0.1:5000을 직접 입력하여 접속해 주세요.

⛔ '웹서버 종료' 버튼: 실행 중인 웹 서버를 안전하게 멈출 때 사용합니다. 프로그램 사용을 다 마쳤을 때 눌러주세요.

📂 '완성된 영상 폴더 열기' 버튼: AI가 만든 모든 영상 파일과 관련 자료들이 저장되는 폴더를 바로 열어줍니다.

⚙️ '고급 설정' 버튼: 음성 설정, 장면 감지 민감도, FFmpeg 경로, GPU 사용 여부 등 좀 더 전문적인 설정들을 변경할 수 있는 창을 엽니다.

🔑 '라이선스 관리' 버튼: 현재 사용 중인 라이선스 키 정보를 확인하고, 필요하다면 새로운 키를 입력할 수 있습니다.

🌐 '도움말' 버튼: 현재는 간단한 안내 메시지만 나오지만, 앞으로 더 자세한 도움말 페이지로 연결될 예정입니다.

상태 표시줄: 창 아래쪽에 현재 서버 상태([⚪] 서버가 중지되었습니다, [⚙️] 서버를 시작하는 중입니다, [🟢] 서버 실행 중!)와 영상 제작 진행 상황이 간략하게 표시됩니다.

🔍 '상세 로그 보기' 버튼: 이 버튼을 누르면 프로그램이 어떤 작업을 하고 있는지 모든 기록(로그)을 볼 수 있는 창이 나타나거나 사라집니다. 문제가 발생했을 때 원인을 파악하는 데 큰 도움이 됩니다.

주의: 이 설정들은 프로그램 작동에 큰 영향을 줄 수 있으니, 프로그램에 익숙해 지기 전까지는 기본값 그대로 사용하는 것을 권장합니다.

✨ 웹 인터페이스 (쇼츠 영상 제작)

메인 제어판에서 '🚀 프로그램 시작 & 웹 UI 열기' 버튼을 누르면, 이제 본격적으로 나만의 쇼츠 영상 제작을 시작할 수 있습니다!

단계별 영상 제작 가이드:

  1. 1. 쇼츠 영상 소스 파일 업로드 🎥 [파일 선택] 버튼을 눌러 AI가 편집할 원본 영상 파일을 골라주세요. 여러 개의 파일을 한 번에 선택해도 괜찮습니다.

    팁: 원본 영상의 품질이 좋을수록 AI가 더 멋진 결과물을 만들어냅니다.

  2. 2. 블러 처리 설정 🌫️ (선택 사항) 영상에 민감한 정보나 특정 인물을 가리고 싶을 때 사용합니다. 업로드된 각 영상 파일 옆의 블러 설정 버튼을 누르면, 팝업 창에서 블러 처리할 영역을 직접 지정하고 강도를 조절할 수 있습니다.

    팁: 블러 처리를 많이 하면 영상 제작 시간이 조금 더 걸릴 수 있습니다.

  3. 3. AI 음성(TTS) 설정 🎤 다양한 한국어 AI 음성 중에서 원하는 목소리를 고르고, 슬라이더로 속도와 음높이를 조절해 보세요. [미리듣기] 버튼으로 바로 확인 가능합니다.
  4. 4. 영상 주제 또는 상품 설명 입력 📝 만들고 싶은 쇼츠 영상의 주제나 소개할 상품에 대한 설명을 상세하게 적어주세요. AI가 이 내용을 바탕으로 대본과 홍보 문구를 만들기 때문에, 구체적이고 매력적으로 작성할수록 더 좋은 결과물을 얻을 수 있습니다.

    예시: "30대 직장인을 위한 효과적인 시간 관리 노하우, 퇴근 후 저녁 시간을 200% 활용하는 꿀팁!"

  5. 5. 장면 전환 민감도 🎬 영상이 장면을 바꾸는 '민감도'를 조절합니다. '세밀하게(낮은 값)'로 하면 짧고 빠르게 장면이 전환되며, '큼직하게(높은 값)'로 하면 길고 안정적인 장면이 유지됩니다. 영상의 분위기에 맞춰 조절해보세요.
  6. 6. 생성할 영상 개수 🔢 AI가 대본과 영상을 조합하여 만들 쇼츠 영상 버전을 1개, 2개, 또는 3개 중에서 선택할 수 있습니다. 다양한 느낌의 영상을 얻고 싶을 때 유용합니다.
  7. 7. 자막 설정 💬 [자막 미리보기 및 설정] 버튼을 눌러 자막 스타일을 바꿀 수 있습니다. 폰트, 크기, 색상, 위치를 자유롭게 조절하고, 미리보기 창에서 직접 드래그하여 원하는 위치에 놓을 수도 있습니다.

    팁: 시청자가 영상을 쉽게 이해하도록 가독성 좋은 자막 설정을 찾는 것이 중요합니다.

  8. 8. 추가 옵션 (마지막 1초 제거) ✂️ [체크]하면 영상의 마지막 1초가 자동으로 잘립니다. 영상 끝에 불필요한 로고나 엔딩 크레딧이 있을 때 활용하세요.

    주의: 이 옵션을 사용하면 영상 제작 시간이 약 2배 정도 더 걸릴 수 있습니다.

  9. 9. 워터마크 적용 💧 [체크]하면 제작되는 모든 영상에 미리 설정된 워터마크 이미지가 자동으로 추가됩니다.
  10. 10. 하드웨어 가속 (GPU) 사용 ⚡ [체크]하면 컴퓨터의 그래픽 카드(GPU)를 활용해 영상 제작 시간을 크게 단축할 수 있습니다. NVIDIA, AMD, Intel GPU를 지원하며, 'AUTO'를 선택하면 최적의 GPU를 자동으로 감지합니다.

    경고: 이 기능은 특정 GPU 드라이버와 FFmpeg 빌드가 필요하며, 잘못 설정하면 문제가 발생할 수 있습니다. 잘 모르는 경우 [체크 해제] 상태(CPU 모드)로 두는 것을 권장합니다.

  11. 11. '쇼츠 영상 생성 시작' 버튼 클릭 ▶️ 모든 설정을 마치셨다면 이 버튼을 누르세요! 잠시 후 결과 확인 페이지로 이동하여 완성된 영상을 볼 수 있습니다.

✅ 결과 확인 페이지

'쇼츠 영상 생성 시작' 버튼을 누르면 결과 확인 페이지로 이동합니다. 여기서 AI가 만든 결과물을 확인하고 다운로드할 수 있습니다.

  • 진행 상황: 페이지 상단에서 'N개의 쇼츠 영상을 생성 중입니다...' 와 같이 현재 작업 진행 상태를 실시간으로 확인할 수 있습니다.
  • 각 버전별 카드: 선택한 영상 개수만큼 카드 형태의 미리보기가 나타납니다. 영상이 완성될 때마다 해당 카드에 미리보기 영상, 썸네일, 대본, 홍보글 다운로드 버튼이 활성화됩니다.
  • 생성된 대본: AI가 만든 영상 대본을 텍스트로 바로 확인할 수 있습니다.
  • SNS 홍보글: 유튜브/틱톡용으로 AI가 작성한 홍보 문구를 [복사하기] 버튼으로 쉽게 복사하여 활용할 수 있습니다.
  • 영상 미리보기: 완성된 영상을 페이지에서 바로 재생하여 확인할 수 있습니다.
  • 영상 썸네일 미리보기: 후킹 멘트가 포함된 썸네일 이미지를 확인 할 수 있습니다.
  • 다운로드: [영상 다운로드] 버튼으로 MP4 영상 파일을, [썸네일 다운로드] 버튼으로 AI가 만든 흥미로운 썸네일 이미지를 받을 수 있습니다.

중요: 이 페이지에서는 기존 영상의 리메이크 기능을 제공하지 않습니다. 다른 설정으로 새로운 영상을 만들고 싶으시면, 웹사이트 상단의 로고를 클릭하여 메인 페이지로 돌아가 새로운 제작을 시작해주세요.

❓ 문제 해결 (Troubleshooting)

프로그램 사용 중 궁금하거나 어려운 점이 있다면 여기서 해결 방법을 찾아보세요.

Q1: 웹 브라우저가 자동으로 열리지 않아요!

원인: 간혹 시스템 설정이나 특정 환경 때문에 브라우저가 자동으로 열리지 않는 경우가 있습니다.

해결책: 메인 제어판의 로그 창에 "[🟢] 서버 실행 중! 웹 브라우저에서 작업을 시작하세요." 메시지가 보인다면, 걱정 마시고 웹 브라우저 주소창에 http://127.0.0.1:5000을 직접 입력하여 접속해주세요.

Q2: 영상 제작 중 'Internal Server Error' 같은 오류가 발생했어요!

원인: 프로그램 내에서 예상치 못한 오류가 발생했을 때 나오는 일반적인 메시지입니다. 정확한 원인은 로그를 봐야 알 수 있습니다.

해결책:

  1. GUI 메인 제어판에서 '상세 로그 보기' 버튼을 눌러 로그 창을 띄웁니다.
  2. 영상 제작 버튼을 누른 후, 로그 창에 'Traceback (most recent call last):' 로 시작하는 오류 메시지가 보이는지 확인합니다.
  3. 이 'Traceback' 내용을 복사해서 프로그램 개발자에게 문의하시면 빠른 해결에 도움이 됩니다.
  4. 자주 발생하는 원인:
    • 영상 파일 문제: 손상되었거나 지원되지 않는 형식의 영상 파일일 수 있습니다. 다른 파일로 다시 시도해보세요.
    • FFmpeg 경로 문제: FFmpeg 경로(FFMPEG_BINARY_PATH)가 정확한지, 해당 경로에 ffmpeg.exe 파일이 실제로 있는지 확인해야 합니다. (고급 설정에서 확인 가능)
    • GPU 가속 설정 오류: '하드웨어 가속'을 사용했는데, 시스템에 맞는 GPU 코덱을 못 찾거나 드라이버 문제가 있을 수 있습니다. '하드웨어 가속' 옵션을 [체크 해제]하고 다시 시도해 보세요.
    • Google Gemini API 키 문제: API 키가 잘못되었거나 만료되었을 수 있습니다.

Q3: 영상 제작 시간이 너무 오래 걸려요.

원인:

  • 원본 영상의 길이가 길거나 해상도가 높을 때 시간이 더 걸립니다.
  • '마지막 1초 제거' 옵션을 사용하면 제작 시간이 두 배 이상 늘어날 수 있습니다.
  • '하드웨어 가속(GPU)'을 사용하지 않거나, GPU 가속이 제대로 활성화되지 않으면 CPU로만 처리되어 시간이 오래 걸립니다.

해결책:

  • 짧은 영상으로 먼저 테스트해 보세요.
  • '마지막 1초 제거' 옵션을 [체크 해제]하고 다시 시도해보세요.
  • 컴퓨터에 GPU가 있고 '하드웨어 가속' 옵션을 [체크]했는데도 느리다면, 고급 설정에서 FFmpeg 경로 및 기본 GPU 타입 설정을 다시 확인해야 할 수 있습니다.

Q4: 생성된 영상의 화질이 안 좋아요.

원인:

  • 원본 영상의 화질이 낮으면 AI도 더 좋은 품질로 만들 수 없습니다.

해결책: 가능한 가장 높은 품질의 원본 영상을 사용해 주세요.

Q5: 자막이 이상하게 보여요.

원인: 선택한 자막 폰트 파일에 문제가 있거나, 설정된 폰트 크기/위치가 영상과 어울리지 않을 수 있습니다.

해결책: '자막 미리보기 및 설정'에서 다른 폰트, 크기, 색상, 위치를 조절하며 테스트해보세요. 특히, 자막을 마우스로 직접 드래그하여 가장 보기 좋은 위치에 놓을 수 있습니다.

Q6: AI 음성이 이상해요.

원인: 사용 중인 tts 서비스에 일시적인 문제가 있거나, 선택한 음성 설정(속도, 음높이)이 대본 내용과 잘 맞지 않을 수 있습니다.

해결책: 'AI 음성(TTS) 설정'에서 다른 음성을 선택하거나, 속도/음높이 슬라이더를 조절한 후 '미리듣기'를 통해 가장 자연스럽고 적절한 목소리를 찾아보세요.
