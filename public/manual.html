AI 쇼츠 팩토리 (VidFactory AI)

프로그램 버전: 1.2.5 (GUI)

🚀 프로그램 소개

'AI 쇼츠 팩토리 (VidFactory AI)'는 인공지능(AI)의 놀라운 힘으로 여러분의 제품이나 아이디어를 소개하는 짧고 멋진 영상(쇼츠)을 쉽고 빠르게 만들어주는 혁신적인 프로그램입니다.

💡 가장 큰 장점: 복잡한 영상 편집 기술이 전혀 없어도 괜찮습니다! 여러분이 제공한 영상과 간단한 설명만 있으면, AI가 알아서 대본을 쓰고, 실감 나는 목소리를 더하고, 필요한 자막까지 척척 만들어 전문가처럼 멋진 쇼츠 영상을 완성해 드립니다.

주요 기능:

  • 📝 AI 자동 대본 & 홍보 문구 생성: 여러분의 아이디어를 AI가 매력적인 영상 대본과 소셜 미디어(유튜브, 틱톡 등) 홍보글로 바꿔줍니다.
  • 🎬 영상 자동 편집: 업로드하신 영상에서 대본에 딱 맞는 최고의 장면들을 자동으로 골라 자연스럽게 연결해 줍니다.
  • 🔊 고품질 AI 음성 더빙 & 자막: 사람처럼 자연스러운 AI 목소리로 대본을 읽어주고, 내용에 딱 맞는 자막을 자동으로 넣어줍니다.
  • ✨ 사용자 맞춤 설정: 자막의 폰트, 크기, 색상, 위치를 자유롭게 조절할 수 있습니다. 중국어 및 특정 영역을 가리는 '블러' 효과 기능도 제공합니다.
  • ⚡ 빠른 영상 제작: 고성능 컴퓨터를 사용하신다면 '하드웨어 가속(GPU)' 기능을 통해 영상 제작 시간을 획기적으로 줄일 수 있습니다.
  • 💻 간편한 사용: 모든 과정은 누구나 쉽게 사용할 수 있는 웹사이트(웹 UI)와 편리한 데스크톱 프로그램(GUI)으로 이루어집니다.
  • ⏱️ 배치 기능: 여러 개의 영상 정보를 한 번에 등록하고, 컴퓨터를 사용하지 않는 시간에 자동으로 영상을 일괄 제작하여 효율적인 시간 관리를 돕습니다.

💡 시작하기 (설치 및 최초 실행)

VidFactory AI는 설치 과정이 아주 간단합니다. 대부분의 설정은 프로그램이 자동으로 잡아주거나, 처음 실행 시 나타나는 안내에 따라 몇 가지만 입력하면 바로 사용할 수 있습니다.

  1. 프로그램 실행: 제공받으신 압축 파일을 풀고 VidFactory_AI.exe 파일을 더블 클릭하여 프로그램을 시작합니다.
  2. (최초 실행) 라이선스 키 입력 🔑: 프로그램을 처음 사용하시거나 라이선스가 만료된 경우, '라이선스 키 입력' 창이 자동으로 뜹니다. 받으신 라이선스 키를 정확히 입력하고 [확인]을 누르면 됩니다. 유효한 키는 만료일과 함께 확인 메시지를 보여주며, 컴퓨터에 안전하게 저장됩니다.

    팁: 라이선스 키는 오타 방지를 위해 반드시 복사(Ctrl+C)하여 붙여넣기(Ctrl+V) 하세요.

    주의: API 키는 외부에 노출되거나 유출되지 않도록 주의해야 합니다.

  3. (최초 실행) Google Gemini API 키 입력 🤖: 라이선스 키 입력 후, Google Gemini API 키가 설정되어 있지 않으면 'Google Gemini API 키 입력' 창이 나타납니다. AI 대본 및 홍보 문구 생성에 꼭 필요한 키이니, Google Cloud Console에서 발급받은 API 키를 입력하고 [저장] 버튼을 누릅니다.

    참고 문서: Google Gemini API 키 획득 방법

    주의: API 키는 외부에 노출되거나 유출되지 않도록 주의해야 합니다.

  4. 메인 제어판 표시: 모든 초기 설정이 완료되면, 'VidFactory AI' 메인 제어판 창이 화면에 나타납니다.

💻 메인 제어판 사용법

메인 제어판은 'VidFactory AI'의 모든 기능을 관리하고, 설정을 바꾸며, 프로그램 상태를 한눈에 파악할 수 있는 중심 허브입니다.

🚀 '프로그램 시작 & 웹 UI 열기' 버튼: 이 버튼을 누르면 'VidFactory AI'의 웹 서버가 켜지고, 인터넷 브라우저가 자동으로 열려 영상 제작 웹사이트로 이동합니다. 서버가 실행 중이면 버튼은 [✅ 프로그램 실행 중]으로 바뀌고 비활성화됩니다. 만약 브라우저가 자동으로 열리지 않으면, 주소창에 http://127.0.0.1:5000을 직접 입력하여 접속해 주세요.

⛔ '웹서버 종료' 버튼: 실행 중인 웹 서버를 안전하게 멈출 때 사용합니다. 프로그램 사용을 다 마쳤을 때 눌러주세요.

📂 '완성된 영상 폴더 열기' 버튼: AI가 만든 모든 영상 파일과 관련 자료들이 저장되는 폴더를 바로 열어줍니다.

▶️ '배치 시작' 버튼: 영상 정보를 일괄 등록한 후, 이 버튼을 눌러 영상 제작을 시작합니다. 컴퓨터를 사용하지 않는 동안 자동으로 영상이 만들어집니다.

📊 '배치 현황판': 현재 진행 중인 배치 작업의 등록, 제작 완료, 실패 현황을 한눈에 파악할 수 있습니다.

🔑 '라이선스 관리' 버튼: 현재 사용 중인 라이선스 키 정보를 확인하고, 필요하다면 새로운 키를 입력할 수 있습니다.

🏠 '홈페이지' 버튼: 프로그램 소개와 사용방법을 볼 수 있습니다.

🌐 '도움말' 버튼: 프로그램 사용 중 자주 하는 질문을 볼 수 있습니다.

상태 표시줄: 창 아래쪽에 현재 서버 상태([⚪] 서버가 중지되었습니다, [⚙️] 서버를 시작하는 중입니다, [🟢] 서버 실행 중!)와 영상 제작 진행 상황이 간략하게 표시됩니다.

🔍 '상세 로그 보기' 버튼: 이 버튼을 누르면 프로그램이 어떤 작업을 하고 있는지 모든 기록(로그)을 볼 수 있는 창이 나타나거나 사라집니다. 문제가 발생했을 때 원인을 파악하는 데 큰 도움이 됩니다.

✨ 웹 인터페이스 (쇼츠 영상 제작)

메인 제어판에서 '🚀 프로그램 시작 & 웹 UI 열기' 버튼을 누르면, 이제 본격적으로 나만의 쇼츠 영상 제작을 시작할 수 있습니다!

단계별 영상 제작 가이드:

  1. 쇼츠 영상 소스 파일 업로드 🎥 [파일 선택] 버튼을 눌러 AI가 편집할 원본 영상 파일을 골라주세요. 여러 개의 파일을 한 번에 선택해도 괜찮습니다.

    팁: 원본 영상의 품질이 좋을수록 AI가 더 멋진 결과물을 만들어냅니다.

    팁: 여러 파일 선택 시 컨트롤 키 누르고 영상 선택하면 됩니다.

  2. 블러 처리 설정 🌫️ (선택 사항) 영상에 민감한 정보나 특정 인물을 가리고 싶을 때 사용합니다. 업로드된 각 영상 파일 옆의 블러 설정 버튼을 누르면, 팝업 창에서 블러 처리할 영역을 직접 지정하고 강도를 조절할 수 있습니다. (최대 5개 가능)

    팁: 블러 처리를 많이 하면 영상 제작 시간이 조금 더 걸릴 수 있습니다.

  3. AI 음성(TTS) 설정 🎤(선택 사항) 다양한 한국어 AI 음성 중에서 원하는 목소리를 고르고, 슬라이더로 속도와 음높이를 조절해 보세요. [미리듣기] 버튼으로 바로 확인 가능합니다.
  4. 초반 후킹 멘트 📝(선택 사항) 영상 초반 후킹 멘트를 사용자가 직접 지정 할 수 있습니다. 없는 경우 AI가 훌륭한 후킹 멘트를 제공합니다.

    예시: 세상에 이런거 봤어?, 이거 모르면 100만원 손해야!

  5. 영상 주제 또는 상품 설명 입력 📝(필수 사항) 만들고 싶은 쇼츠 영상의 주제나 소개할 상품에 대한 설명을 상세하게 적어주세요. AI가 이 내용을 바탕으로 대본과 홍보 문구를 만들기 때문에, 구체적이고 매력적으로 작성할수록 더 좋은 결과물을 얻을 수 있습니다.

    예시: 30대 직장인을 위한 효과적인 시간 아이템, 모던한 디자인, 기능 소개

    주의: 상품 설명란에 이모티콘, 이모지는 입력하지 마세요. 영상 제작 실패의 원인이 됩니다.

  6. 구매방법 (SNS 홍보 게시물에 사용) 이 정보는 AI가 생성한 "SNS 홍보 게시물(댓글 유도, 상품 번호 포함)"에 자동으로 삽입됩니다. 구매 유도를 위한 상품 정보 및 유도 멘트를 적어주세요.
  7. 장면 전환 민감도 🎬(선택 사항) 영상이 장면을 바꾸는 '민감도'를 조절합니다. '세밀하게(낮은 값)'로 하면 짧고 빠르게 장면이 전환되며, '큼직하게(높은 값)'로 하면 길고 안정적인 장면이 유지됩니다. 영상의 분위기에 맞춰 조절해보세요.
  8. 자막 설정 💬(선택 사항) [자막 미리보기 및 설정] 버튼을 눌러 자막 스타일을 바꿀 수 있습니다. 폰트, 크기, 색상, 위치를 자유롭게 조절하고, 미리보기 창에서 직접 드래그하여 원하는 위치에 놓을 수도 있습니다.

    팁: 시청자가 영상을 쉽게 이해하도록 가독성 좋은 자막 설정을 찾는 것이 중요합니다.

  9. 추가 옵션 (마지막 1초 제거) ✂️(선택 사항) [체크]하면 영상의 마지막 1초가 자동으로 잘립니다. 영상 끝에 불필요한 로고나 엔딩 크레딧이 있을 때 활용하세요.

    주의: 이 옵션을 사용하면 영상 제작 시간이 더 걸릴 수 있습니다.

  10. 썸네일 설정 💬(선택 사항) 썸네일 이미지에 나오는 자막 위치와 스타일을 바꿀 수 있습니다. 폰트, 크기, 색상, 위치를 자유롭게 조절하고, 미리보기 창에서 직접 드래그하여 원하는 위치에 놓을 수도 있습니다.

    팁: 시청자가 영상을 쉽게 이해하도록 가독성 좋은 자막 설정을 찾는 것이 중요합니다.

  11. 하드웨어 가속 (GPU) 사용 ⚡ [체크]하면 컴퓨터의 그래픽 카드(GPU)를 활용해 영상 제작 시간을 크게 단축할 수 있습니다. NVIDIA, AMD, Intel GPU를 지원하며, 'AUTO'를 선택하면 최적의 GPU를 자동으로 감지합니다.

    정보: 하드웨어 가속시 영상 제작 시간 : 평균 2 ~ 4분, 미 사용 시 3 ~ 6분 정도 소요됩니다. * 영상 길이와 자막 길이에 따라 변동 있을 수 있습니다.

  12. '쇼츠 영상 즉시 제작 시작' 버튼 클릭 ▶️ 모든 설정을 마치셨다면 이 버튼을 누르세요! 잠시 후 결과 확인 페이지로 이동하여 완성된 영상을 볼 수 있습니다.
  13. '담아두기 (배치 작업 추가)' 버튼 클릭 ▶️ 입력된 설정값을 배차 작업에 추가합니다. 배치 작업 추가는 여러번 등록 가능 합니다.(배치 작업 중에는 즉시 생성 작업이 불가능 합니다.), 배치 작업 시작은 메인프로그램으로 돌아거서 "배치시작" 버튼을 누르면 시작합니다.

✅ 결과 확인 페이지

'쇼츠 영상 즉시 제작 시작' 버튼을 누르면 결과 확인 페이지로 이동합니다. 여기서 AI가 만든 결과물을 확인하고 다운로드할 수 있습니다.

  • 진행 상황: 페이지 상단에서 '쇼츠 영상을 생성 중입니다...' 와 같이 현재 작업 진행 상태를 실시간으로 확인할 수 있습니다.
  • 생성된 대본: AI가 만든 영상 대본을 텍스트로 바로 확인할 수 있습니다.
  • SNS 홍보글: 유튜브/틱톡용으로 AI가 작성한 홍보 문구를 [복사하기] 버튼으로 쉽게 복사하여 활용할 수 있습니다.
  • 썸네일 포함된 결과 영상 미리보기: 썸네일이 포함된 완성된 영상을 페이지에서 바로 재생하여 확인할 수 있습니다. 다운로드 후 sns에 업로드 가능합니다.
  • 영상 미리보기: 썸네일이 없는 완성된 영상을 페이지에서 바로 재생하여 확인할 수 있습니다. 다운로드 후 sns에 업로드 가능합니다.
  • 영상 썸네일 미리보기: 후킹 멘트가 포함된 썸네일 이미지를 확인 할 수 있습니다. 다운로드 후 sns에 업로드 가능합니다.

✅ 배치 현황 페이지

'배치 시작' 버튼을 누르면 배치 현황 페이지로 이동하며, 대량 영상 제작의 효율을 극대화할 수 있습니다.

  • 배치 워커 상태 : 등록된 배치 수, 대기 중, 진행 중, 완료, 실패 정보를 한 눈에 볼 수 있습니다.
  • 대기 중인 작업 : 작업 전 대기 중인 작업 정보를 표시합니다.
  • 현재 진행 중인 작업 : 현재 진행 중인 작업 정보가 표시 됩니다.
  • 완료된 영상(갤러리) : 작업이 끝난 영상 정보가 표시 됩니다. 다운로드 후 sns에 업로드 가능 합니다.
  • 실패한 작업: 실패한 작업과 실패 이유가 표시 됩니다.(실패한 작업은 영상 제작 횟수가 차감 되지 않습니다)

❓ 문제 해결 (Troubleshooting)

프로그램 사용 중 궁금하거나 어려운 점이 있다면 여기서 해결 방법을 찾아보세요.

Q1: 웹 브라우저가 자동으로 열리지 않아요!

원인: 간혹 시스템 설정이나 특정 환경 때문에 브라우저가 자동으로 열리지 않는 경우가 있습니다.

해결책: 메인 제어판의 로그 창에 "[🟢] 서버 실행 중! 웹 브라우저에서 작업을 시작하세요." 메시지가 보인다면, 걱정 마시고 웹 브라우저 주소창에 http://127.0.0.1:5000을 직접 입력하여 접속해주세요.

Q2: 영상 제작 중 'Internal Server Error' 같은 오류가 발생했어요!

원인: 프로그램 내에서 예상치 못한 오류가 발생했을 때 나오는 일반적인 메시지입니다. 일반적으로 컴퓨터 재부팅 후 해결됩니다.

Q3: 영상 제작 시간이 너무 오래 걸려요.

원인:

  • 원본 영상의 길이가 길거나 해상도가 높을 때 시간이 더 걸립니다.
  • '마지막 1초 제거' 옵션을 사용하면 제작 시간이 두 배 이상 늘어날 수 있습니다.
  • '하드웨어 가속(GPU)'을 사용하지 않거나, GPU 가속이 제대로 활성화되지 않으면 CPU로만 처리되어 시간이 오래 걸립니다.

해결책:

  • 짧은 영상으로 먼저 테스트해 보세요.
  • '마지막 1초 제거' 옵션을 [체크 해제]하고 다시 시도해보세요.
  • 컴퓨터에 GPU가 있고 '하드웨어 가속' 옵션을 [체크]했는데도 느리다면, 고급 설정에서 FFmpeg 경로 및 기본 GPU 타입 설정을 다시 확인해야 할 수 있습니다.

Q4: 생성된 영상의 화질이 안 좋아요.

원인:

  • 원본 영상의 화질이 낮으면 AI도 더 좋은 품질로 만들 수 없습니다.

해결책: 가능한 가장 높은 품질의 원본 영상을 사용해 주세요.

Q5: 자막이 이상하게 보여요.

원인: 선택한 자막 폰트 파일에 문제가 있거나, 설정된 폰트 크기/위치가 영상과 어울리지 않을 수 있습니다.

해결책: '자막 미리보기 및 설정'에서 다른 폰트, 크기, 색상, 위치를 조절하며 테스트해보세요. 특히, 자막을 마우스로 직접 드래그하여 가장 보기 좋은 위치에 놓을 수 있습니다.

Q6: AI 음성이 이상해요.

원인: 사용 중인 tts 서비스에 일시적인 문제가 있거나, 선택한 음성 설정(속도, 음높이)이 대본 내용과 잘 맞지 않을 수 있습니다.

해결책: 'AI 음성(TTS) 설정'에서 다른 음성을 선택하거나, 속도/음높이 슬라이더를 조절한 후 '미리듣기'를 통해 가장 자연스럽고 적절한 목소리를 찾아보세요.
