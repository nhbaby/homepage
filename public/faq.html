❓ 자주 묻는 질문 (FAQ)

💡 Q1: AI 쇼츠 팩토리(VidFactory AI)는 어떤 프로그램인가요?

사용자가 제공한 영상과 제품 설명을 바탕으로, AI(Google Gemini)가 대본, 음성, 자막을 자동으로 생성하고, 영상 편집까지 수행하여 빠르고 쉽게 쇼츠(Shorts) 형식의 홍보 영상을 만들어주는 프로그램입니다.

💻 Q2: 이 프로그램을 사용하려면 무엇을 설치해야 하나요?

기본적으로 제공받으신 VidFactory AI GUI 프로그램만 실행하면 됩니다.

💰 Q3: 프로그램은 무료인가요?

  1. 무료 사용권 (영상 제작 횟수 10회, 기간 3일) : 비용 무료
  2. 크리에이터 사용권 (영상 제작 횟수 100회, 기간 30일) : 비용 ₩44,900/월
  3. 프로 사용권(영상 제작 횟수 500회, 기간 30일) : 비용 ₩149,000/월

🔑 Q4: Google Gemini API 키는 무엇이며, 왜 필요한가요?

Google Gemini API 키는 AI 쇼츠 팩토리의 핵심 기능인 AI 대본 및 홍보 문구 생성을 위해 Google의 인공지능 서비스에 접근하는 열쇠입니다. 이 키가 없으면 AI 기능을 사용할 수 없습니다. 최초 실행 시 입력 요청이 뜨며, `고급 설정`이나 `라이선스 관리` 메뉴를 통해서도 다시 입력할 수 있습니다.

🎥 Q5: 어떤 종류의 영상을 업로드할 수 있나요?

MP4, MOV 등 대부분의 일반적인 비디오 형식을 지원합니다. 하지만 영상 파일이 손상되었거나 매우 특이한 코덱을 사용하는 경우 문제가 발생할 수 있습니다. 가급적 표준적인 고화질 MP4 파일을 권장합니다. (* 이미지 파일은 지원하지 않습니다.)

🎬 Q6: '장면 전환 민감도'는 무엇인가요? 어떻게 조절해야 하나요?

업로드한 영상에서 새로운 장면을 선택하는 기준을 말합니다.

  • 낮은 민감도(세밀하게): 영상의 작은 움직임에도 장면을 자주 바꿔 짧은 장면들을 많이 활용합니다.
  • 높은 민감도(큼직하게): 큰 변화가 있을 때만 장면을 바꿔 길고 안정적인 장면들을 주로 사용합니다.

영상의 흐름이나 강조하고 싶은 내용에 따라 조절해 보세요.

⚡ Q7: '하드웨어 가속 (GPU)'은 무엇이며, 꼭 사용해야 하나요?

여러분의 컴퓨터에 장착된 그래픽 카드(GPU)의 성능을 활용하여 영상을 더 빠르게 제작하는 기능입니다. 필수는 아니지만, GPU가 있고 이 기능을 활성화하면 영상 제작 시간을 크게 단축할 수 있습니다. 하지만 특정 드라이버나 FFmpeg 버전이 필요하므로, 잘 모른다면 기본값(비활성화)으로 두는 것이 좋습니다.

⚠️ Q8: 영상 제작 중 'Internal Server Error'와 같은 오류가 발생했어요.

프로그램 웹 서버에서 처리할 수 없는 오류가 발생한 것입니다. 일반적으로 컴퓨터 재부팅후 증상이 해결 됩니다.

⏳ Q9: 영상 제작 시간이 너무 오래 걸려요.

영상 길이나 화질, 선택한 옵션(`마지막 1초 제거`, `블러 처리`)에 따라 시간이 오래 걸릴 수 있습니다. 컴퓨터에 GPU가 있다면 '하드웨어 가속 (GPU)' 옵션을 사용해 보세요.

🌟 Q10: 생성된 영상의 화질이 좋지 않거나 자막이 이상해요.

원본 영상의 화질이 낮으면 결과물도 좋을 수 없습니다. 고화질 원본 영상을 사용해 보세요. 자막이 이상하다면 웹 UI의 `자막 설정`에서 폰트, 크기, 색상, 위치를 다시 조절해보고 미리보기로 확인해 보세요.

🌐 Q11: 웹 브라우저가 자동으로 열리지 않습니다.

GUI 메인 제어판에서 `상태 표시줄`에 `[🟢] 서버 실행 중!` 메시지가 뜬다면, 웹 서버는 정상적으로 작동하고 있는 것입니다. 웹 브라우저 주소창에 http://127.0.0.1:5000을 직접 입력하여 접속할 수 있습니다.

📂 Q12: 생성된 영상은 어디에 저장되나요?

GUI 메인 제어판에서 `📂 완성된 영상 폴더 열기` 버튼을 누르면, 제작된 모든 영상 파일들이 저장된 폴더가 바로 열립니다.
