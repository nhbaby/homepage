# firebase 배포 하는 방법
- 설치 : npm install -g firebase-tools
- 프로젝트 폴더로 이동
- firebase 로그인 : firebase login
- firebase 초기화: firebase init
- 질문 : 프로젝트 선택, public 선택, single-page no, github no
- public 폴더로 이동
- 배포 : firebase deploy --only hosting
